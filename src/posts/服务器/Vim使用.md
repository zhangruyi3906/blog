# Vim的基本使用

常用命令：
查找字符串，例如搜索user：`/user`，此时关键字被高亮，按n可以找下一个匹配，N为上一个



| 用法               | 键位                       |
| ------------------ | -------------------------- |
| 移到单词开头       | b                          |
| 移到单词结尾       | e                          |
| 撤回               | u                          |
| 取消撤回           | control+r                  |
| 删除一行           | dd                         |
| 向后删一个单词     | de                         |
| 不退出vim执行shell | `:shell`,然后exit返回vim   |
| 搜索,例如user      | `/user`，然后n往下找,N往上 |
| 取消搜索高亮       | `:noh`                     |
| 移动到行尾         | `^`也就是`shift+6 `        |
| 移动到行首         | `$`,也就是`shift+4`        |
| 删除到结尾         | `d$`                       |
| 替换一个字母       | r                          |
|                    |                            |
