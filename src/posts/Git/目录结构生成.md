# 目录结构生成

现在想要把开源在GitHub上面的项目生成一份目录结构，如下 ：

![image.png](https://s2.loli.net/2024/04/28/DF9eNdWqwlYp372.webp)

### 安装

使用tree工具,Mac安装：
```sh
brew install tree
```


### 使用

```sh
tree -L 深度 目录 
```

例如
```sh
tree -L 1
```

![image.png](https://s2.loli.net/2024/04/28/hx2FAVQbyTO6mDX.webp)


忽略某些文件：
```sh
tree  -I node_modules -I .vuepress -L 4
```
