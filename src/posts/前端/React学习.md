# React

## 创建项目

```shell
npx create-react-app react-learning --template typescript
```

其中的`react-learning`是项目名字

启动项目：

```shell
cd  react-learning
npm start
```

修改默认端口：

在src下面新建一个.env.development的文件,里面写上

```shell
PORT=7070
```
